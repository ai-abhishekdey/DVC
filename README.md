# DVC

This repository contain learning related to Data Version Control (DVC)

## Getting Started:

### Create virtual environment

```
conda create -n dvc_env python=3.10 -y

conda activate dvc_env

```

### Install DVC

```
pip3 install dvc[all]

```

### Initialize 


```

git init

dvc init

```
* **Note:**  **dvc init** should be done in repository root not insids folders
